CSS概述

发表于 2016-02-27 | 修改于 2016-02-19   |   分类于 CSS系列

样式表

样式表是随着HTML的发明而出现的，不同的浏览器有着各自的不同的样式语言,读者使用样式语言来调节网页的显示方式。

  • 样式表是HTML的发明的产物
  • 样式表的形式和样式语言是多种多样的。

CSS

层叠样式表，Cascading Style Sheets的缩写，是一种样式表语言，用来描述HTML，XML文档的呈现。CSS描述怎么在屏幕，电子纸，音频，和其他媒体上渲染结构化元素。

CSS的重要内容：

  • CSS文件的加载过程和解析
  • 浏览器的默认样式
  • CSS样式来源和层叠规则
  • CSS的选择器和权重及集成(优先级)
  • 浏览器的页面呈现
  • CSS布局
