浏览器默认样式

  • 浏览器默认样式概括
    • 浏览器有默认的样式，来对没有外部样式的页面进行渲染
    • 不同的浏览器的默认样式是不同的。
    • 各浏览器默认样式对照表
  • 默认样式的理解
    浏览器内核是不会规定标签默认的样子，标签的默认样式是由浏览器的默认样式所决定的，在浏览器对CSS进行解析生成CSS Tree的时候，加入进去的。

重置浏览器样式

  • 为了保证页面在不同浏览器中显示的尽可能的一致，我们会重置不同浏览器默认的样式，称为重置样式表。
  • 重置样式表资源
    • CSS Reset
    • strppd.css
    • normalize.css
