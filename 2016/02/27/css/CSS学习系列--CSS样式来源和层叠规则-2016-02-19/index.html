CSS样式来源和层叠规则

发表于 2016-02-27 | 修改于 2016-02-20   |   分类于 CSS系列

CSS样式来源

  • 开发者样式(高)
    • 内联样式(高)
      网页作者直接在HTML的标签中写CSS信息。
    • 内部样式表(中)
      网页作者在HTML中通过<style>标签来写CSS信息。
    • 外部样式表(低)
      网页作者在HTML中通过<link>标签来引入的外部的独立css文件。
  • 读者样式(中)
    网页的读者可以在浏览器内部设立一个css文件，可以对浏览器开启的所有HTML文件有效。一般就是更改字体大小和样式。
  • 浏览器样式(低)
    浏览器拥有一套自己的默认样式，当没有外部样式的时候，可以使用默认样式来显示HTML。不同的浏览器有不同的默认样式。

CSS层叠

  • CSS的层叠
    • CSS的层叠指的是浏览器对多个样式的来源进行叠加，最终确定结果的过程。CSS之所以有“层叠”的概念，是因为有多个样式来源。层叠是CSS的核心机制，理解了它才能以最经济的方式写出最容易改动的CSS，让文档在达到设计要求的同时，也给用户留下一些空间，让他们根据需要更改文档的显示效果。
    • CSS的层叠指的是对不同来源中相同的样式进行的叠加，即只有样式名一致的样式才能进行层叠。
  • CSS的层叠规则
    根据样式的优先级，浏览器在加载样式的时候会计算出最终的样式值。样式值的生成有叠加和覆盖两种方式。
    • CSS样式的优先级
      开发者样式 > 读者样式 > 浏览器样式
    • CSS叠加规则
      不同样式来源的样式中，独有的样式属性进行叠加合并。
    • CSS覆盖规则
      不同样式来源的样式中，相同的样式属性根据样式来源的优先级进行覆盖。
    • 更改样式来源中样式的优先级
      在样式属性后使用 !important 来强行改变属性的优先级为最高。
